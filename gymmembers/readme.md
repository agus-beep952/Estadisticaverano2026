# 🏋️‍♂️ Conjunto de Datos de Miembros de Gimnasio

## 📘 Descripción General
Este conjunto de datos ofrece una visión detallada de las **rutinas de ejercicio, atributos físicos y métricas de condición física** de los miembros de un gimnasio.  
Contiene **973 muestras**, incluyendo indicadores clave de rendimiento como **frecuencia cardíaca**, **calorías quemadas** y **duración del entrenamiento**.  
Cada registro también incluye **datos demográficos** y **niveles de experiencia**, lo que permite un análisis integral de patrones de ejercicio, progresión atlética y tendencias de salud.

---

## 🔑 Características Principales

| Variable | Descripción |
|-----------|--------------|
| **Age** | Edad del miembro del gimnasio. |
| **Gender** | Género del miembro (*Masculino* o *Femenino*). |
| **Weight (kg)** | Peso del miembro en kilogramos. |
| **Height (m)** | Altura del miembro en metros. |
| **Max_BPM** | Frecuencia cardíaca máxima (latidos por minuto) durante las sesiones de entrenamiento. |
| **Avg_BPM** | Frecuencia cardíaca promedio durante las sesiones. |
| **Resting_BPM** | Frecuencia cardíaca en reposo antes del entrenamiento. |
| **Session_Duration (hours)** | Duración de cada sesión de entrenamiento en horas. |
| **Calories_Burned** | Total de calorías quemadas en cada sesión. |
| **Workout_Type** | Tipo de entrenamiento realizado (*Cardio, Strength, Yoga, HIIT*). |
| **Fat_Percentage** | Porcentaje de grasa corporal del miembro. |
| **Water_Intake (liters)** | Consumo diario de agua durante los entrenamientos. |
| **Workout_Frequency (days/week)** | Número de sesiones de entrenamiento por semana. |
| **Experience_Level** | Nivel de experiencia, desde *principiante (1)* hasta *experto (3)*. |
| **BMI** | Índice de Masa Corporal, calculado a partir de la altura y el peso. |

---

## 🎯 Aplicaciones
Este conjunto de datos es ideal para:
- **Científicos de datos** que analizan la progresión física y tendencias de salud.  
- **Investigadores en salud** que estudian hábitos de ejercicio y respuestas fisiológicas.  
- **Entusiastas del fitness** que exploran la relación entre demografía y métricas de rendimiento.  

---

## 📊 Posibilidades de Análisis
Con su amplia gama de variables, el dataset permite explorar:
- Cómo **edad, género y composición corporal** afectan la intensidad del entrenamiento.  
- La relación entre **nivel de experiencia** y **calorías quemadas**.  
- Patrones en el **comportamiento de la frecuencia cardíaca** y la **resistencia** según el tipo de entrenamiento.  

---

## ⚙️ Usos Potenciales
- Modelado predictivo de la **mejora física** a lo largo del tiempo.  
- Agrupación de miembros según **estilo o intensidad de entrenamiento**.  
- Análisis de correlación entre **BMI**, **Fat_Percentage** y **Calories_Burned**.  

---

## 🧠 Resumen
Este conjunto de datos constituye una base sólida para **análisis estadístico**, **aprendizaje automático** y **estudios de optimización de la salud**, ofreciendo valiosas perspectivas sobre cómo las variables de estilo de vida y entrenamiento interactúan para moldear el rendimiento físico.
